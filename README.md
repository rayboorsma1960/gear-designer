## Features

### 5 Gear Types
- **Spur Gear** — Straight-cut teeth for simple power transmission
- **Helical Gear** — Angled teeth for smoother, quieter operation
- **Herringbone Gear** — Double helical for zero axial thrust
- **Bevel Gear** — Conical gears for perpendicular shaft connections
- **Worm Gear** — High reduction ratio worm and wheel pairs

### ISO 54 Standard Sizes
- 13 standard module sizes: 0.5, 0.8, 1.0, 1.25, 1.5, 2.0, 2.5, 3.0, 4.0, 5.0, 6.0, 8.0, 10.0 mm
- 3 standard pressure angles: 14.5, 20, 25 degrees
- Customizable tooth count, face width, and helix angle

### 3D Print Ready
- Export watertight manifold STL files — no repair needed
- Tested and verified on Bambu Lab printers
- Proper bore holes for shaft mounting

### Interactive 3D Preview
- Real-time 3D visualization powered by Three.js
- Orbit, zoom, and pan controls
- Meshing gear pair animation
- Dimension readout panel with all gear specifications

### Cross-Platform
- Works on **Windows**, **macOS**, and **Linux**
- Fully responsive on **Android** and **iOS** devices
- No install, no signup — just open and design

## How to Use

1. **Open the app** — [rayboorsma1960.github.io/gear-designer](https://rayboorsma1960.github.io/gear-designer/)
2. **Select a gear type** from the dropdown
3. **Choose your module size** (ISO 54 standard)
4. **Adjust parameters** — tooth count, face width, pressure angle, helix angle
5. **Preview in 3D** — rotate and inspect your gear
6. **Export STL** — click the export button and print!

## Technical Details

- Single HTML file — no dependencies, no build step, no server required
- Three.js for 3D rendering
- Involute tooth profile mathematics
- Watertight manifold geometry for reliable 3D printing
- ASCII STL export
